# 📸 Instagram Scraper + Analytics

Este proyecto es un scraper de Instagram desarrollado en Python utilizando cookies de sesión para autenticación.

Permite extraer información de perfiles públicos, analizar contenido y almacenar los datos en formato JSON.

---

## 🚀 Funcionalidades

- 🔐 Autenticación mediante cookies (sessionid, csrftoken)
- 📊 Extracción de información de perfiles
- 📸 Obtención de posts (imágenes y videos)
- 🖼️ Descarga automática de imágenes
- 🎥 Detección de tipo de contenido (imagen/video)
- 📝 Extracción de captions completos
- 🏷️ Análisis de hashtags (#)
- 👤 Análisis de menciones (@)
- 👥 Extracción de seguidores
- 💾 Exportación de datos en formato JSON

---

## 🧠 Tecnologías utilizadas

- Python 3
- Instaloader
- JSON
- Regex (expresiones regulares)
